ATALHOS PARA PROGRAMADOR

  • Home e End

    voltam no inicio e vão para o final, retrospectivamente

  • ctrl + setas

    move o cursor de palavra em palavra

  • ctrl + shift + setas

    serve para selecionar, com o cursor indo de palavra em palavra

  • ctrl + delete

    serve para apagar uma palavra inteira

  • ctrl + ;

    utilizado para deixar como tudo que foi selecionado, independente da linguagaem de programação mas variavel conforme o editor de texto

  • alt + setas para cima e para baixo

    serve para mover uma linha inteira com código para cima ou para baixo

  • shift + tab

    utilizado para corrigir a identação para esquerda

  • shift + home

    serve para selecionar uma linha inteira com o cursor estando no final dela

  • shift + end

    serve para selecionar uma linha inteira com o cursor estando no início dela
